# Contest Title Normalization Audit

Rules applied:
- `Olympia Amateur` -> `Amateur Olympia`
- lowercase `of`
- no periods
- remove `Pro Qualifier`
- remove `Open` for `NPC Worldwide` contests
- keep `Open` in `IFBB` contests
- preserve `MuscleContest` camel case
- if consolidation creates multiple same-year events, append `(2)`, `(3)`, etc.
